# 重活一世 - 详细需求规格说明书

## 1. 项目概述

### 1.1 项目名称

**重活一世** (Live Another Life)

### 1.2 项目类型

Web H5 人生模拟游戏

### 1.3 核心概念

以新中国成立（1949年）为起点至今的人生模拟器。用户选择出生日期后，随机分配角色身份，在半个月（真实时间）的游戏过程中体验完整人生。

### 1.4 目标用户

- 对中国近现代史感兴趣的用户
- 喜欢人生模拟/文字冒险游戏的玩家
- 怀旧情结的中老年用户
- 年轻用户对祖辈生活的好奇

***

## 2. UI/UX 规格

### 2.1 整体布局

- **居中展示**: 最大宽度 480px，模拟手机竖屏体验
- **安全适配**: 两侧留白 16px，整体居中
- **全屏体验**: 内容区域全屏，顶部状态栏透明

### 2.2 视觉风格

- **主色调**: 暖色调怀旧风

  - 辅色: #D4A574 (驼色)
  - 强调色: #C41E3A (中国红)
  - 背景: #F5F0E6 (米黄)
  - 文字: #2C1810 (深褐)
- **字体**:
  - 标题: "ZCOOL XiaoWei", "STKaiti", serif
  - 正文: "Noto Serif SC", serif
  - 数字: "Playfair Display", serif
- **视觉效果**:
  - 纸张纹理背景
  - 复古边框装饰
  - 轻微阴影营造层次感
  - 流畅的页面过渡动画

### 2.3 组件设计

#### 卡片组件

- 圆角: 12px
- 阴影: 0 4px 20px rgba(139, 69, 19, 0.15)
- 内边距: 20px
- 背景: rgba(255, 255, 255, 0.9)

#### 按钮组件

- 主按钮: 中国红背景，白色文字
- 次按钮: 透明背景，褐色边框
- 圆角: 8px
- 悬停效果: 轻微上移 + 阴影加深

#### 进度条

- 高度: 8px
- 圆角: 4px
- 渐变填充: 从深褐到浅驼

## 3. 功能规格

### 3.1 核心流程

```
[开始页面] → [日期选择] → [角色生成] → [人生模拟] → [结束总结]
```

### 3.2 页面详情

#### 3.2.1 开始页面

- 游戏标题展示（毛笔字风格）
- 副标题：简短介绍
- "开始新人生" 按钮
- 历史背景简介

#### 3.2.2 出生日期选择

- 年份选择器: 1949 - 2024
- 月份选择器: 1 - 12
- 日期选择器: 随月份变化
- 显示当日历史事件提示
- 确认按钮

#### 3.2.3 角色生成

- 随机roll点机制
- 显示角色属性:
  - 身份: 农民、工人、学生、教师、医生、商人、公务员等
  - 性格: 内向/外向/中庸
  - 天赋: 随机3项特长
- 家庭背景: 城市/农村、富裕/普通/贫困
- 出生地: 随机省份城市

#### 3.2.4 人生模拟主界面

**顶部状态栏**:

- 当前日期 (年/月/日)
- 年龄显示
- 地点显示
- 生命值/健康值

**主体区域**:

- 今日事件描述 (主要文本区域)
- 天气状况 + 情感状态图标
- 行为标签 (如: 工作、学习、娱乐)

**底部操作区**:

- 继续明天按钮
- 跳过按钮 (可选择跳过n天/n年)
- 大事记按钮

#### 3.2.5 大事记页面

- 时间线展示
- 按年份分类
- 可添加个人感悟
- 可标记重要事件 ★

#### 3.2.6 跳过功能

- 弹出层选择
- 跳过天数: 1/7/30/365
- 跳过年份: 1/5/10
- 确认跳过

#### 3.2.7 人生结束

- 总结一生
- 成就展示
- 统计信息 (经历事件数、寿命、巅峰时期等)
- 再次重活按钮

### 3.3 数据系统

#### 角色身份库 (50+)

- 农民、工人、学生、教师、医生、护士
- 工程师、科学家、军人、警察、公务员
- 商人、司机、厨师、服务员、个体户
- 艺术家、作家、记者、主持人

#### 地区数据

- 省份: 34个 (含直辖市)
- 城市: 每省3-5个代表性城市
- 街道/村庄: 随机生成

#### 大事件库 (200+)

- 政治类: 建国、土改、反右、文革、改革开放等
- 经济类: 三大改造、大跃进、市场经济、WTO等
- 社会类: 人口政策、城镇化、下岗潮、网购等
- 文化类: 样板戏、恢复高考、留学潮、网络时代等
- 科技类: 两弹一星、载人航天、互联网、移动支付等
- 国际类: 抗美援朝、中苏关系、改革开放、奥运等

#### 天气系统

- 晴、雨、雪、阴、多云
- 极端天气: 台风、洪水、干旱

#### 情感状态

- 快乐、平静、焦虑、悲伤、愤怒、惊讶

## 4. 技术架构

### 4.1 技术栈

- Vue 3 + Composition API
- Element Plus 组件库
- Tailwindcss 样式库
- Vite 构建工具
- 纯前端，无后端依赖

### 4.2 数据存储

- LocalStorage 存储游戏进度
- 大事记数据本地保存

### 4.3 性能要求

- 首屏加载 < 2s
- 页面切换流畅
- 离线可用
